os :: indicam as pseudo-classes

que nesse caso está sendo utilizada para colocar "coisas" antes ou depois de cada link:

        
            a::after{
                content: '[link]';
                text-decoration: none;
                font-weight: normal;
                color: gray;
            }

se você quiser que um link apareça diferente, colcoa uma classe nele, como eu coloquei "especial" e dá um tratamento diferenciado a ela

simbolos importantes

seletores personalizados

        
        #  = id
        .  = class
        :  = pseudo-class
        :: = pseudo-elements
        >  = children
